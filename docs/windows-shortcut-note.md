# Windows ショートカットとファイル関連付けガイド

このドキュメントでは、特定のPythonアプリケーション（`moleditpy.exe`）に対し、特定の拡張子（**.pmeraw**）のファイルを関連付け、ダブルクリックで開けるようにする手順と、使用するアイコンについて説明します。

## 1\. 実行ファイルと関連付けるファイル

| 項目 | 詳細 |
| :--- | :--- |
| **実行ファイル** (Target Application) | `C:\Users\%USERNAME%\AppData\Local\Programs\Python\PythonXX\Scripts\moleditpy.exe` |
| **関連付けたい拡張子** (File Extension) | **`.pmeraw`** |

> **プレースホルダーの置き換え:**
>
>   * `%USERNAME%` は、お使いの**Windowsのユーザー名**に置き換えてください。
>   * `PythonXX` は、インストールされている**Pythonのバージョンディレクトリ名**（例: `Python313` や `Python311` など）に置き換えてください。

## 2\. 使用するアイコン (Icon)

ファイル関連付けやショートカットに使用するアイコンファイルは以下を指定します。

| 項目 | 詳細 |
| :--- | :--- |
| **アイコンファイルのパス** | `C:\Users\%USERNAME%\AppData\Local\Programs\Python\PythonXX\Lib\site-packages\moleditpy\assets\icon.ico` |

-----

## 3\. ファイルの関連付け手順 (Windows)

**`.pmeraw`** ファイルをダブルクリックで `moleditpy.exe` で開けるようにするには、以下の手順で**ファイルの関連付け**を行います。

### ステップ 1: 任意の `.pmeraw` ファイルを見つける

エクスプローラーで、関連付けを行いたい拡張子を持つファイル（例: `sample.pmeraw`）を**右クリック**します。

### ステップ 2: 関連付けの変更

1.  コンテキストメニューから **「プロパティ」** を選択します。
2.  「全般」タブの **「ファイルの種類」** の横にある **「変更...」** ボタンをクリックします。

### ステップ 3: プログラムの選択

1.  「このファイルを開く方法を選んでください」というウィンドウが表示されたら、**「その他のアプリ」** → **「PCでアプリを探す」** を選択します。
2.  以下のパス（プレースホルダーを実際のパスに置き換えたもの）にある **`moleditpy.exe`** を指定します。
    ```
    C:\Users\%USERNAME%\AppData\Local\Programs\Python\PythonXX\Scripts\moleditpy.exe
    ```
3.  **「開く」** をクリックします。

### 補足: アイコンの設定

カスタムアイコンをシステム全体のファイル関連付けに適用するには、通常、**レジストリを編集**する必要があります。レジストリ操作は上級者向けであり、慎重に行ってください。

-----

## 4\. デスクトップショートカットの作成手順

アプリケーションを起動するためのショートカットを作成し、アイコンを設定する手順です。

### ステップ 1: 新規ショートカットの作成

1.  デスクトップなどの任意の場所を**右クリック**します。
2.  **「新規作成」** → **「ショートカット」** を選択します。

### ステップ 2: 実行ファイルのパスを入力

1.  「項目の場所を入力してください」に以下のパス（プレースホルダーを実際のパスに置き換えたもの）を入力します。
    ```
    C:\Users\%USERNAME%\AppData\Local\Programs\Python\PythonXX\Scripts\moleditpy.exe
    ```
2.  **「次へ」** をクリックします。

### ステップ 3: ショートカットの名前を指定

ショートカットに任意の名前（例: `moleditpy`）を付けて、**「完了」** をクリックします。

### ステップ 4: アイコンの変更

1.  作成されたショートカットを**右クリック**し、**「プロパティ」** を選択します。
2.  「ショートカット」タブの **「アイコンの変更...」** ボタンをクリックします。
3.  以下のパスにある **`icon.ico`** を参照して選択します。
    ```
    C:\Users\%USERNAME%\AppData\Local\Programs\Python\PythonXX\Lib\site-packages\moleditpy\assets\icon.ico
    ```
4.  **「OK」** → **「適用」** → **「OK」** をクリックして完了です。

-----

## 5\. ショートカットをスタートメニューに配置する方法

Windowsのスタートメニューは通常、特定の隠しフォルダに保存されています。作成したアプリケーションのショートカットをスタートメニューのプログラム一覧に表示するには、以下のフォルダにショートカットをコピーまたは移動します。

### ステップ 1: スタートメニューフォルダのパスを確認

エクスプローラーのアドレスバーに以下のいずれかのパスを入力し、Enterキーを押します。

#### A. 現在のユーザー専用のスタートメニュー (推奨)

このフォルダに配置すると、ログイン中のユーザーのみのスタートメニューに表示されます。

```
%APPDATA%\Microsoft\Windows\Start Menu\Programs
```

#### B. すべてのユーザー共通のスタートメニュー (管理者権限が必要な場合あり)

このフォルダに配置すると、PCのすべてのユーザーのスタートメニューに表示されます。

```
%ALLUSERSPROFILE%\Microsoft\Windows\Start Menu\Programs
```

### ステップ 2: ショートカットのコピーまたは移動

1.  ステップ 1 で開いたスタートメニューフォルダのウィンドウと、作成したショートカット（例: デスクトップ上の `Mole EditPy` ショートカット）があるウィンドウを並べます。
2.  ショートカットをマウスでドラッグし、スタートメニューフォルダのウィンドウにドロップして**移動**するか、`Ctrl`キーを押しながらドラッグして**コピー**します。

### ステップ 3: 確認

スタートボタンをクリックし、プログラム一覧からショートカットに付けた名前（例: `moleditpy`）が追加されていることを確認してください。

-----

これで、デスクトップショートカットの作成からスタートメニューへの配置までが完了しました。
